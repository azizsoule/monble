## Features

Main features of the app are :

- Use the app in offline mode ✅
- Autenticate with passcode ⏳
- Authenticate with fingerprint or faceID ⏳
- Track incomes ⏳
- Track expenses ⏳
- Create budgets ⏳
- Track budget overruns ⏳
- See a daily / monthly / yearly report of your finance ⏳
